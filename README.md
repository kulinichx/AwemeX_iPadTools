# AwemeX iPad Tools

## V1.4.0

新增 DYYY 风格“面板设置”选项：

- 启用新版长按面板
- 长按面板玻璃效果
- 长按面板深色模式
- 保存面板玻璃效果
- 面板毛玻璃透明度
- 长按面板保存视频
- 长按面板保存封面
- 长按面板保存音频
- 长按面板保存图片
- 长按保存所有图片
- 长按面板生成视频
- 长按面板复制文案

说明：

- 选项已经接入 NSUserDefaults 并实时保存。
- 长按菜单会根据开关显示项目。
- 复制文案已做基础实现。
- 保存视频/封面/音频/图片/生成视频的真实保存动作，需要继续 hook 抖音或 DYYY 原保存接口后才能完整生效。

## 编译

有根：

```bash
make clean package
```

无根：

```bash
make clean package SCHEME=rootless
```
