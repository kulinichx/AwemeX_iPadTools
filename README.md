# AwemeX V9 RightContainerScale

基于 V8_RightButtonsSafeScope 继续开发。

## V9 核心变化

- 右侧按钮缩放从“逐个元素 transform”改为“右侧整列容器 transform”。
- 右侧按钮透明度仍沿用 V8 的逐项启发式识别，保证透明度控制稳定。
- 新增右侧容器识别：靠右、中下区域、窄宽度、高度较高、内部至少形成 3 个纵向按钮特征点。
- 新增容器评分逻辑，优先选择更靠右、高度更像整列按钮、宽度不过大的候选容器。
- 新增上一次缩放容器弱引用，候选变化时自动清理旧 transform。
- anchorPoint 只调整一次，避免每秒 timer 反复应用导致轻微漂移。

## 测试建议

安装后杀掉抖音重新打开。测试右侧按钮缩放 0.8x / 1.0x / 1.2x，重点观察：

1. 右侧按钮间距是否保持自然；
2. 顶部推荐/关注/搜索是否不被缩放；
3. 评论/分享/收藏/头像是否跟随整列一起缩放；
4. 切换视频后缩放是否仍能保持。
